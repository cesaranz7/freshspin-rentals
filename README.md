# FreshSpin Rentals

Marketing site for FreshSpin Rentals — a washer & dryer rental service serving Fort Worth and the West Dallas area.

## Live Site
- Live URL: https://freshspin-rentals.vercel.app/

## Features
- Homepage with service area and pricing overview
- Pricing page with tiered monthly rates
- Contact page linking to Facebook Messenger (placeholder link for now)
- Shared header navigation across pages

## Pricing
**Value (Older Units)**
- $50 / month — Washer & Dryer set
- $35 / month — Single machine

**Premium (Newer Units)**
- $65 / month — Washer & Dryer set
- $40 / month — Single machine

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Git + GitHub
